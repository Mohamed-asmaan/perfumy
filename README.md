# Perfumy 🌸

A simple React-based perfume product UI showcasing a clean layout with navigation, search bar, product listing, and footer.

---

## 🚀 Features

- 🧭 Navbar with navigation links
- 🔍 Search bar UI
- 🛍️ Product listing section
- ℹ️ About section
- 📞 Footer with contact details
- 🎨 Basic styling with CSS

---

## 🛠️ Built With

- React (Create React App)
- HTML5
- CSS3
